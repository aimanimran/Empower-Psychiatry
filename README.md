# Empower Psychiatry Website

Official website for [Empower Psychiatry](https://empowerpsychiatryil.com), a complete-care psychiatry practice based in Bloomingdale, Illinois.

## Overview

This website was built to provide a modern, accessible, and SEO-friendly experience for patients and families seeking psychiatric care.

The site highlights:

* Practice philosophy and holistic care approach
* Psychiatric services offered
* Patient testimonials
* Educational mental health resources
* Appointment and contact information
* Mobile-friendly responsive design
